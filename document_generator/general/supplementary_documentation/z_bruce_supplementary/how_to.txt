# ------------------------------------ VIM ---------------------------------------

ma  # sets a local mark named "a" at the current cursor position (line and column) within the current file.
mA  # sets a global mark named "A" that can be jumped to from any file. 
'a  # (apostrophe + letter) jumps to the beginning of the line where mark "a" was set.
`a  # (backtick + letter) jumps to the exact line and column where mark "a" was set.
'A  # or `A will jump to the specified global mark, even opening the correct file if you are in a different buffer. 



# Replace all instances of "static" with "user" on the current and next 2 lines
:,+2s/static/user/g

# Jump to last cursor position
ctrl + o

# Copy lines 16-18 and return to starting cursor position and paste (note "^" is how ctrl is printed in the vim terminal)
:normal! 16G 3yy ^O p


# pasting: 
p           #(Put after): Pastes character-wise text after the cursor or line-wise text on a new line below.
P           #(Put before): Pastes character-wise text before the cursor or line-wise text on a new line above.
:<N>put     # Pastes below line <N>


# Yank current word
yiw

# -> Replace current word with last word yanked
viwp

# repeat last action:
word modification: "."
substitution: "&"

# Delete without placing in clipbard: Use the "black hole" register!  it is "_", and, like 
# all register specification in normal mode, you must start the command with a double quote.
# ex: delete word under cursor into the black hole register:
"_diw

# Jump to next occurance of symbol on current line (capitalize the letter for previous)
f{char}
# Previous
F{char}
# until char before next
t{char}
# until char just after previous
T{char}

# Yank from cursor to symbol, inclusive.  (Then "p" to paste wherever cursor is)
yf{char}   ex:  yf]   yanks from cursor to ], inclusive
# Same but exclusive
yt{char}

# Same as above, but for deleting (capitalize the jump command (f,t) to go backwards)
df{char}
dt{char}      (usually more useful, for example if deleting between quotations)

# Move cursor half page down/up
ctrl + d    /    ctrl + u

# Move 5 lines down/up
5 j      /      5 k

# Enter insert mode at position after cursor: a
# Enter insert mode at end of line: A
# Enter insert mode at beginning of line: I

# delete current word, leaving whitespace intact: diw
# delete current word, and any single space after (or before if none after): daw
# delete from cursor to end of word, including following space: dw
# delete from cursor to end of word, but not the following space: de
# substidue "c" in the above 4 commands to enter insert mode aftewards

# create new line above, enter insert mode: O
# create new line below, enter insert mode: o

# forward one word: w
# forward one word to end of next word: e
# back one word: b
# back one word to end of previous word: ge

# macros/recording
q a   to start recording in register a,   the q again to stop recording
@ a   to execute macro in register a, or
@@ to repeat last recorded macro

To apply macro @a to lines 3-6, use:
:3,6normal! @a

:g/pattern/normal! @a executes the macro on all lines containing the specified pattern. 

# delete forawards
d$ (or c$ to enter insert mode) or D (or C), respectively

# delete backwards 
( I think you can substitude "c" for "d" to delete and enter insert mode)
d0 from cursor to begining of line, excluding character under cursor
dv0 from cursor to begining of line, including character under cursor
x deletes character under
X deletes character immediately before cursor
dh deletes characture under corsor and moves ... haven't figured these out
db deletes backward to beginning of current word, not including character under cursor 
dvb deletes backward to beginning of current word, including character under cursor 


# Join line below with current line
J while on current line
gJ same but without added space
5J join current line with the next 4 lines
:j or :join same
:-j join previous line with current line

:4y   (short for :4yank)

:%s/pattern//gn

# ------------------------------------ BASH ---------------------------------------

# Print all files in current directory which do NOT match the given search string
find . -maxdepth 1 -type f ! -name "*.tex" -print
# delete all files in current directory which do NOT match the given search string
find . -maxdepth 1 -type f ! -name "*.tex" -delete

# grep show only first match in each file
grep -m 1 "pattern" filename
